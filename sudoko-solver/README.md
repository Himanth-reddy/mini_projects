# Sudoku Solver

A simple Sudoku solver project using a Python backend and a web UI.

## Tech

- Python
- Flask
- HTML
- JavaScript

## Run

```bash
pip install flask requests
python app.py
```

Open: `http://127.0.0.1:5000`

## Files

- `app.py` - Flask backend and Sudoku solving logic
- `index.html` - frontend UI
